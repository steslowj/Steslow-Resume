Houseplants vs Fertility Rate in U.S.A

Geog 572 路 Lab 1
UW-Mad Graphic Design in Cartography
  • Summary
  • Proj Header
  • Purchasers
  • Bivariate
  • Plants Purchased

The assignment in full is a longform infographic. I chose to use vertical space to facilitate storytelling, and thus my project does not display well in a portfolio. Look through the tabs to see parts of my infographic, and click the little leaf icon to open a new tab with the full project.

In this infographic, I dig into research on current houseplant purchasers, using data from a large plant industry report. I tried to balance giving an accurate picture of houseplant data with overwhelming use of graphs. Data was analyzed and combined in Excel and drawn up in Illustrator. Geographic data was processed in ArcGIS Pro.

This is the header and beginning of my infographic. I give a brief history of houseplants to start the story of plants vs fertility rate.

One of the more interesting pieces of data from the houseplant report was a look into what life stages that plant purchasers value. I felt it was important to show what was valued, and what was not.

The infographic includes a bivariate choropleth map of houseplant interest vs fertilty rate decrease. Neither of these topics are straightforward, but it resulted in an interesting geographic comparison.

This project taught me I really enjoy reading about color theory, and that there is a very specific subset of data scientists that enjoy writing about color schemes.

Although my topic was semi-serious, the format of infographic allowed me to add some fun to the graphs. For this graphic, I found icons that balance visually with the chart and illustrate each category.

Style Practice

Colored Pencils

ESRI styles "Sketch"

ESRI has dozens of artistic styles (downloadable elements to add into ArcGIS Pro projects). Many of them are robust and easy to use, which makes it great fun to practice more artistic cartography. This style uses colored pencils on countour lines.

Fantasy Ink Cartography

ESRI styles "My Precious"

This is using a style inspired by fantasy novels with large, dense, artistic world maps: Tolkien. The inky linework worked well applied to a city park, but I think the style is best suited for areas with varied terrain.

Degree Progress Tracking in Excel X

Personal project
  • Class List
  • Deadlines
  • Calendar

This is a personal project I started for use in tracking my GIS Degree progress. I wanted something useful but also something fun to work in. The base spreadsheet is from a Microsoft Template online, but I added functionality to show progress via conditional formatting. This project had me learning both data processing and design formatting with Excel.

Information on the Class List page is available to populate the Deadlines page.

Information on the Deadlines page is automatically added to the Calendar page. This has some very nuanced date calculation and conditional formatting built into the Template - but I was able to expand the calendar into more months and modify the styling for my needs.

Ghibli MapBox Studio Style

Geog 572 路 Lab 3
UW-Mad Graphic Design in Cartography
  • Globe View
  • Japan
  • Italy

Mapbox allows users to create custom styles to geographic and political data in the application "Mapbox Studio." This is a style I made inspired by the animated fantasy films by Studio Ghibli.

The leaf-icon link on this card opens a new tab with an interactive version of this style.

I tried to emulate an animated feel with the saturated colors. I also added icson showing places that inspired Ghibli films (many in Japan, yes).

The style has some hillshade coloring and a lot of colors based on terrain. I wanted to both show beauty in the landscape and highlight cultural areas - this view shows metro areas in pink polygons.
